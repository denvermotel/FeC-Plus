FeC-Plus — Guida

Due modi d'uso: l'interfaccia grafica (GUI) e la riga di comando (fec_cli.py), senza GUI.

Requisiti

Python 3.12 e le dipendenze del progetto.

pip install -r requirements.txt
# solo se si usa il backend «browser»:
playwright install chromium

Interfaccia grafica (GUI)

Il modo più semplice: doppio click sul launcher, o avvio da terminale.

  • Windows: avvia_fec.bat  ·  macOS: avvia_fec.command
  • Versione sviluppatore (Test Login, cattura HAR, scelta backend/headless): avvia_fec_dev.bat / avvia_fec_dev.command
  • Oppure: python fec_gui.py

Cartelle di download (Impostazioni)

In ⚙ Impostazioni si imposta la cartella di download di default e, se serve, una cartella diversa per ogni tipo di documento.

  • Per impostazione predefinita tutti i download vanno in un'unica cartella, in sottocartelle per tipo e codice fiscale (es. FattureEmesse_<CF>).
  • Spuntando «personalizza» accanto a una classe (fatture emesse, ricevute, transfrontaliere, massive, corrispettivi, bolli) si assegna a quel tipo una cartella dedicata.
  • L'opzione «senza sottocartella» (per ciascun tipo) salva i file direttamente nella cartella scelta, senza creare la sottocartella: utile per importarli nelle directory dei gestionali di contabilità.

Riga di comando (fec_cli.py)

Stesso motore della GUI, senza interfaccia. Fa il login e scarica passando tutto come argomenti.

Sintassi

python fec_cli.py [ARGOMENTI DI ACCESSO] COMANDO [ARGOMENTI DEL COMANDO]

Argomenti di accesso (comuni a tutti i comandi)

ArgomentoDescrizione
--cfCF Fisconline o nome utente Entratel. Obbligatorio.
--pinPIN. Obbligatorio.
--passwordPassword. Obbligatorio (oppure --password-env).
--password-env VARLegge la password dalla variabile d'ambiente VAR (consigliato).
--cfstudioCF dello studio incaricante (profili 1 e 2).
--cf-clienteCF del cliente delegante.
--pivaP.IVA dell'utenza di lavoro (richiesta da massive e bolli).
--profilo {1,2,3}1 = studio→cliente · 2 = cassetto studio · 3 = me stesso. Default: 1.
--backend {requests,browser}Backend di login. Default: requests (nessun browser).
--no-headlessMostra la finestra del browser (solo backend browser).
--dest DIRCartella di destinazione dei download. Default: ./Download.
--dry-runMostra cosa verrebbe eseguito, senza fare login né download.

Comandi

ComandoCosa scaricaArgomentiFormato date
emesseFatture emesse--dal --alggmmaaaa
ricevuteFatture ricevute--dal --al [--tipo-data 1|2]ggmmaaaa
transfrontaliere-emesseTransfrontaliere emesse--dal --alggmmaaaa
transfrontaliere-ricevuteTransfrontaliere ricevute--dal --alggmmaaaa
messe-a-disposizioneFatture messe a disposizione--dal --alggmmaaaa
massive-emesseRichiesta massiva emesse--dal --alaaaa-mm-gg
massive-ricevute-emissioneMassiva ricevute (data emissione)--dal --alaaaa-mm-gg
massive-ricevute-ricezioneMassiva ricevute (data ricezione)--dal --alaaaa-mm-gg
corrispettiviRichiesta massiva corrispettivi--dal --alaaaa-mm-gg
bolliBolli virtuali → PDF F24--trimestre 1|2|3|4 --anno aaaa—

--tipo-data: 1 = ricerca per data ricezione (default), 2 = per data emissione.

⚠️ Sicurezza: le credenziali passate come argomenti restano visibili nella cronologia della shell e nell'elenco dei processi. Preferire --password-env per non scrivere la password in chiaro nel comando. La password non viene comunque mai salvata su disco.

Esempi

# Fatture emesse di gennaio 2026 (studio → cliente)
python fec_cli.py --cf RSSMRA80A01H501U --pin 1234 --password segreta \
    --cfstudio 01234567890 --cf-cliente 09876543210 \
    emesse --dal 01012026 --al 31012026

# Corrispettivi 1° trimestre, password da variabile d'ambiente
set FEC_PWD=segreta
python fec_cli.py --cf RSSMRA80A01H501U --pin 1234 --password-env FEC_PWD \
    --cf-cliente 09876543210 --piva 09876543210 \
    corrispettivi --dal 2026-01-01 --al 2026-03-31

# Bolli virtuali 1° trimestre 2026 → PDF F24
python fec_cli.py --cf RSSMRA80A01H501U --pin 1234 --password-env FEC_PWD \
    --cf-cliente 09876543210 --piva 09876543210 \
    bolli --trimestre 1 --anno 2026

# Vedere cosa verrebbe eseguito, senza login
python fec_cli.py --cf X --pin Y --password Z --dry-run emesse --dal 01012026 --al 31012026

Codici di uscita

CodiceSignificato
0Operazione completata.
1Errore durante il download (es. sessione scaduta).
2Login fallito (credenziali/PIN errati o argomenti mancanti).
